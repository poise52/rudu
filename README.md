# rudu

Терминальный просмотр занятого места на диске в стиле ncdu: один раз просканировал дерево от выбранной папки, дальше листаешь каталоги без повторного обхода.

Собирается под **Linux** и **macOS** (нужны Unix-метаданные для размеров в блоках и разделения по файловым системам).

## Сборка

Нужен [Rust](https://rustup.rs/) (stable, желательно свежий — в проекте `edition = "2024"`).

```bash
git clone https://github.com/poise52/rudu.git
cd rudu
cargo build --release
```

Бинарник: `target/release/rudu`.

## Запуск

Без аргумента смотрит **текущую директорию**:

```bash
./target/release/rudu
```

С путём (как с ncdu):

```bash
./target/release/rudu ~/Downloads
./target/release/rudu .
```

Из исходников без установки бинарника:

```bash
cargo run --release
cargo run --release -- ~/Documents
```

После `--` идут аргументы самой `rudu`, иначе Cargo их съест.

Справка:

```bash
./target/release/rudu --help
```

## В программе

Первый заход — полный скан от корня сессии (в заголовке спиннер). Потом навигация из памяти.

| Клавиша | Действие |
|---------|----------|
| ↑ / ↓ | выбор строки |
| Enter | зайти в каталог |
| Backspace | на уровень выше |
| r | пересканировать дерево от корня сессии |
| q | выход |
